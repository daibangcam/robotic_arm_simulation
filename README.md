# H2Q Lab - Robotic_Arm_Simulation

- Dự án mã nguồn mở để tính động học thuận, động học nghịch và mô phỏng động học của các loại cánh tay Robot cơ bản trong môn học "Kỹ thuật Robot"
- Hy vọng các đồng nghiệp giảng viên sẽ dễ dàng minh họa trực quan bài giảng hơn.
- Hy vọng các bạn sinh viên sẽ dễ hiểu bài và có nhiều đam mê hơn trong ngành học khô khan này.

## Notes
- create_ico.py: tạo file .ico để làm icon cho phần mềm
- build_exe.py: đóng gói chương trình thành file .exe để chạy độc lập
- main.py: chương trình chính
